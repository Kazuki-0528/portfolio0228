# TICKET-010 モバイルメニュー動作修正

## 依頼内容（背景・目的・期限）
SPでメニュー開閉後にaria-expandedが更新されない不具合。目的は操作性と支援技術対応改善。期限: 2026-02-11。

## 影響範囲（ページ/コンポーネント）
mock-site/assets/js/main.js と全ページのナビボタン属性。

## 対応方針
クリック時の開閉クラス・aria-expandedを同期し、キーボード操作でも確認。

## 完了条件（チェック項目）
- 主要ブラウザ: Chrome / Edge / Safari で表示確認
- 画面幅: 320px / 375px / 768px / 1280px で確認
- リンク: 内部リンク・外部リンクの遷移確認
- フォーム: 入力・必須・送信動作（contact対象チケット以外は影響有無）
- 表示崩れ: 見出し、余白、画像比率、表組みの崩れ確認
